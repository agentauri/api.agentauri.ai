Welcome to the **AgentAuri API Documentation**. AgentAuri provides real-time blockchain monitoring infrastructure for the AI agent economy, built on the [ERC-8004](https://eips.ethereum.org/EIPS/eip-8004) standard.

## What is AgentAuri?

AgentAuri transforms blockchain events from Identity, Reputation, and Validation registries into automated actions:

- **Telegram Notifications** - Get instant alerts when agents are registered or receive feedback
- **REST Webhooks** - Integrate with your existing systems via HTTP callbacks
- **MCP Server Updates** - Power AI agent workflows with real-time data
